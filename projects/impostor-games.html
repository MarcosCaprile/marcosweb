Impostor Games

Multiplayer-Partyspiel · Local Multiplayer · Web für Schul-iPads

Worum geht’s?

Impostor Games ist eine Sammlung kleiner Social-Deduction- und Party-Minispiele für lokalen Multiplayer. Ziel war eine Webversion, die auch auf beschränkten Schul-iPads gut läuft.

Design-Entscheidungen

  • „One Device“-Flow: ein Bildschirm, mehrere Spieler
  • Kurze Runden, klare Regeln
  • UI so simpel, dass niemand Erklärungen braucht

Was ich daran mag

  • Game-Design trifft Produktdenken
  • UX in Stresssituationen (Zeitdruck, Gruppe, Spaß)
  • Technisch: stabil & performant im Browser

Status

Fertig als Webversion – Aktuell jedoch Offline gestellt.
