So testest du den Simulator lokal

  1. Python installieren
    Empfohlen: Python 3.11 oder 3.12.
  2. ZIP entpacken
    Öffne den entpackten Ordner in VS Code oder im Terminal.
  3. Abhängigkeiten installieren
    Im Terminal im Projektordner: 
    python -m pip install --upgrade pip
pip install numpy matplotlib
  4. Programm starten
    Datei ausführen (Name ggf. anpassen): 
    python universe_simulator.py
  5. Wenn kein Fenster erscheint / Interaktion fehlt
    Dann fehlt oft ein GUI-Backend. Auf Windows/macOS meist ok – ansonsten: 
    # optional (Linux)
sudo apt-get install python3-tk
    Danach erneut starten.
Controls: Drag = Rotation · Wheel = Zoom · Follow-Body = Kamera folgt Objekt · Slider = Speed (Tage/Step)
